Publications

  • 2024.06
    Testing the Effect of Code Documentation on Large Language Model Code Understanding
    North American Association for Computational Linguistics (NAACL) 2024
    This research empirically analyzes how code documentation quality impacts the code generation and understanding capabilities of Large Language Models (LLMs). It reveals that incorrect documentation significantly hinders LLMs' code comprehension, while incomplete or missing documentation has no significant impact.
  • 2023.11
    Reinforcement Learning for Wildfire Mitigation in Simulated Disaster Environments
    Neural Information Processing Systems (NeurIPS) 2023
    This paper presents a reinforcement learning approach to wildfire mitigation in simulated disaster environments. We release two software libraries, SimFire and SimHarness, to facilitate future research in this area. The former is a wildfire simulator, and the latter is the reinforcement learning library.
  • 2022.09
    Practical Attacks on Machine Translation using Paraphrase
    Association for Machine Translation in the Americas (AMTA) 2022
    This research investigated the vulnerability of machine translation systems to adversarial attacks constructed with limited information. A novel attack method was proposed that generates perturbations using paraphrases and evaluates their impact on meaning preservation and translation degradation across various language pairs and systems.
  • 2021.04
    The vulnerability of UAVs: an adversarial machine learning perspective
    SPIE
    This research proposes a methodology to evaluate the vulnerability of unmanned aerial vehicles (UAVs) to adversarial machine learning attacks by analyzing potential attack vectors at each stage of UAV operation and assessing the feasibility of both established and novel attack methods.
